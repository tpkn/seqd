# seqd

Just like Linux `seq`, but for date.

## Usage

```
seqd [-Y | -M | -D | -h | -m | -s] "<start_date>" "<end_date>"
```

## Options

```
-Y           Step by years
-M           Step by months
-D           Step by days
-h           Step by hours
-m           Step by minutes
-s           Step by seconds
--help       Help
--version    Version
```
