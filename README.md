## Usage

```
seqd [-Y | -M | -D | -h | -m | -s] [-r] "<start_date>" "<end_date>"
```

## Options

```
<end_date>   Can also be a "eom" (end of month) or "eoy" (end of year)
-Y           Step by years
-M           Step by months
-D           Step by days
-h           Step by hours
-m           Step by minutes
-s           Step by seconds
-r           Dates in reverse order
--help       Help
--version    Version
```
